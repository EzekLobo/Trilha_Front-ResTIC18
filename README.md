## Visão Geral

Trilha_Front-ResTIC18 é um projeto voltado ao desenvolvimento web com foco em arquitetura organizada, operações CRUD, integração com serviços externos e exibição dinâmica de dados.

Principais pontos:

- Landing Page centralizada  
- CRUD completo  
- Sincronização em tempo real via Firebase  
- Interfaces ricas para gerenciamento de dados  
- Componentes e rotas modulares
